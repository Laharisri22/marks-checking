# 📊 Marks Checking System

The **Marks Checking System** is a simple web-based application that allows users to enter subject marks and automatically calculate the **total marks, average score, and grade**. This project helps students quickly evaluate their academic performance.

---

## 🚀 Features

* 📝 Enter marks for multiple subjects
* ➕ Automatically calculate **total marks**
* 📊 Calculate **average marks**
* 🏆 Display **grade based on marks**
* ⚡ Instant result display
* 📱 Simple and user-friendly interface

---

## 🛠️ Technologies Used

* **HTML** – Structure of the application
* **CSS** – Styling and layout
* **JavaScript** – Logic for calculating marks, average, and grade

---

## ⚙️ How It Works

1. The user enters marks for different subjects.
2. Click the **Calculate button**.
3. The system calculates:

   * Total marks
   * Average marks
   * Grade
4. The result is displayed instantly on the screen.
